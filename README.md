# MEFENSTEIN

DOOM tarzında Python ile geliştirilmiş birFirst Person Shooter (FPS) oyunu.

## Özellikler

- Raycasting kullanarak First-person 3D görüntüleme
- Farklı silah türleri
- Yapay zeka ile düşman davranışları
- Ses efektleri ve müzik
- Can ve mermi toplama sistemi
- İnteraktif menü sistemi

## Gereksinimler

- Python 3.x
- Pygame

## Kurulum

```bash
# Depoyu klonlayın
git clone https://github.com/KULLANICI_ADINIZ/MEFENSTEIN.git

# Bağımlılıkları yükleyin
pip install pygame
```

## Nasıl Oynanır

Oyunu başlatmak için:
```bash
python main.py
```

Kontroller:
- WASD - Hareket
- Mouse - Etrafı kontrol etme
- Sol Tık - Ateş etme
- 1,2 - Silah değiştirme
- ESC - Duraklat/Menü
